В тази лекция ще кликнем върху връзката, която видяхме в предната лекция.
Тя ни изпраща на уебсайт, създаден от `Microsoft`, който е относно тип-ниво изявленията.
Тук ни се казва, че `Console Application with C#` генерира такива изявления.
Какво се случва тук?
След пускането на `C# 6` преди няколко години, `Microsoft` добави този коментар във `Visual Studio`.  Ако сега създаваме конзолно приложение, имаме възможност да премахнем това, като просто маркираме квадратче.
Нека да видим кое квадратче се има предвид. Ще направим нов проект.

![](../../Assets/7.png)

Предният път ние не маркирахме това квадратче. Ако сега го направим, ще получим следното.

> [!code] Топ-ниво изявления - документация
> ```csharp
> namespace ConsoleApp1
> {
> 	 internal class Program
> {
> 		 static void Main(string[] args)
> 	{
> 	     Console.WriteLine("Hello, World!");
> 	      }
> 	 }
> }
> ```

Тук се случват доста повече неща. Виждаме, че има доста повече код, но единствения, който знаем, е само на един ред. То е същото като това, което ползвахме в последния ни проект. Около него сега има други неща.
Откъде обаче идва целия този код?
В миналото това беше необходимо, за да е стартира което и да е конзолно приложение или код.
Не сме написали нито един ред код, а те просто бяха добавени в нашата програма и сега можем да ги използваме.
Тук се случват много неща и ние няма да навлизаме в подробности за тях точно сега.
